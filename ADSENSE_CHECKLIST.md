# AdSense申請前チェックリスト（このサイト用）

対象: `Appleの領収書のスクショにタイトルを自動でつけるサイト`

## 1. サイト基本要件

- [ ] 本番URLが `https://` で公開されている
- [ ] 独自ドメインで公開している（推奨）
- [ ] モバイル表示で崩れない
- [ ] ページ読み込みが極端に遅くない
- [ ] 404ページがある（最低限）

## 2. 法務ページ

- [x] プライバシーポリシーがある（`privacy.html`）
- [x] 利用規約がある（`terms.html`）
- [ ] お問い合わせ導線がある（メール or フォーム）
- [ ] 必要に応じて運営者情報ページを追加
- [ ] `Apple非公式/非提携` の明記がトップから確認できる

## 3. 個人情報・同意

- [ ] Cookie利用について説明を明記
- [ ] 同意バナー（CMP）を導入
- [ ] 同意拒否時の広告挙動を確認
- [ ] 「画像を保存しない」説明と実装の整合を確認

## 4. コンテンツ品質

- [ ] ツールの使い方説明が十分（誰が見ても使える）
- [ ] サイトの目的が明確
- [ ] コピー中心ではなく独自コンテンツがある
- [ ] 免責・注意事項が読める位置にある

## 5. 広告配置ルール

- [ ] ボタンの近くに誤クリック誘導になる広告配置をしない
- [ ] ドロップ領域の直近に紛らわしい広告を置かない
- [ ] モバイルで画面を広告が覆いすぎない
- [ ] 「ダウンロード」と誤認される位置に広告を置かない

## 6. 商標・ブランド表現

- [ ] Appleロゴ画像は使わない
- [ ] 公式サービスと誤認される表現を避ける
- [ ] タイトル/説明に「非公式ツール」を明記

## 7. 申請前の実動作確認

- [ ] 画像ドロップが安定動作
- [ ] OCR失敗時のフォールバック動作
- [ ] ZIP一括保存が動作
- [ ] Appleプリセット（会社名/日付/合計/サービス名）が安定
- [ ] 主要ブラウザで確認（Chrome/Safari）

## 8. 申請時に用意する情報

- [ ] Googleアカウント
- [ ] サイトURL
- [ ] 支払い先情報
- [ ] 住所/本人確認情報（後続工程で必要）

## 9. 申請後の運用

- [ ] ポリシー違反通知メールを監視
- [ ] UI更新時に広告位置の再確認
- [ ] 法務ページの最終更新日を更新
- [ ] 不具合報告窓口を運用
